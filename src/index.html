Help

Adding Elements

State: Double click anywhere.

Edge: Click on origin state, then click on destination state.

Alternating edge: Click on origin state, hold shift, then click on destination states.

Empty start: Shift+double click anywhere.

Modifying elements

State/Edge

Open editing menu: Double click on state/edge.

Modify acceptance sets: Click to select or deselect acceptance sets.

Modify label: Left click to select cursor position for insertion position, right click to delete. A keyboard is presented after clicking on the label input field.

Sidebar

Atomic propositions

Adding: Click add button

Removing: Click “X” next to the atomic proposition you want to remove. Atomic propositions that are currently being used in the automaton can not be removed.

Alias

Adding: Click add button.

Alias key: Key can only contain letters, numbers, “_” and “-”. Alias key can not be changed if alias is currently being used in the automaton.

Alias value : Alias value is inputted the same way as labels.

Removing: Click “X” next to the alias you want to remove. Aliases that are currently being used in the automaton can not be removed.
